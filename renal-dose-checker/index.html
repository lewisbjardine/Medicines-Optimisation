Renal Dose Calculator

Prototype tool – highlights dose adjustments by eGFR.
Always check dosing in BNF / SPC / Renal Drug Handbook / SPS / local guidance.

Search for a medicine

Enter the most recent eGFR (or CrCl if using Cockcroft–Gault). If left blank, dosing bands will be shown without a highlighted recommendation.

Type at least 2 letters to see suggestions.

Important:
• This is a simplified decision-support prototype.
• Always verify doses in BNF, SPS, SPC, Renal Drug Handbook & local guidelines.
• DOACs must be dosed using creatinine clearance, not eGFR.
• Consider AKI, frailty, drug interactions and acute illness (“sick day rules”).
